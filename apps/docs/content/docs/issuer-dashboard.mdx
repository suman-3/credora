# Issuer Dashboard

![Issuer Dashboard](https://images.unsplash.com/photo-1460925895917-afdab827c52f?w=1200&h=600&fit=crop&crop=center)

## Overview

The Issuer Dashboard is the command center for verified institutions and organizations to issue certificates, manage recipients, and track their certification activities. It provides powerful tools for bulk operations, analytics, and user management.

## Dashboard Layout

### Navigation Sidebar
- **Dashboard Home**: Overview and statistics
- **Issue Certificate**: Create new certificates
- **User Search**: Find and manage recipients
- **Issued Certificates**: View all issued certificates
- **Templates**: Manage certificate templates
- **Analytics**: Detailed reporting and insights
- **Settings**: Organization preferences
- **Team Management**: Manage team members

### Main Content Area
- Real-time statistics and metrics
- Quick action buttons
- Recent activity feed
- Pending tasks and notifications

## Core Features

### 1. User Search

![User Search](https://images.unsplash.com/photo-1450101499163-c8848c66ca85?w=1000&h=600&fit=crop&crop=center)

**Search Functionality:**
- **Wallet Address Search**: Primary search method
- **Email Search**: Search by recipient email
- **Name Search**: Find users by display name
- **Advanced Filters**: Filter by registration date, activity, etc.

**Search Process:**
1. Enter search criteria (wallet address, email, or name)
2. Click "Search" button
3. **Success**: User details displayed
4. **Failure**: "User not found" message with options to:
   - Try different search terms
   - Invite user to platform
   - Return to dashboard

**User Information Display:**
- **Basic Profile**: Name, email, profile picture
- **Wallet Address**: Verified blockchain identity
- **Join Date**: When user registered on platform
- **Certificate History**: Previous certificates from your organization
- **Verification Status**: Account verification level

**Actions Available:**
- **Issue Certificate**: Direct certificate issuance
- **View History**: All certificates issued to this user
- **Send Message**: Contact user directly
- **Add to Favorites**: Quick access for frequent recipients

### 2. Certificate Issuance

![Certificate Issuance](https://images.unsplash.com/photo-1434030216411-0b793f4b4173?w=1000&h=600&fit=crop&crop=center)

**Issuance Form Fields:**
- **Recipient**: User wallet address (from search)
- **Certificate Title**: Degree, course, or achievement name
- **Description**: Detailed certificate description
- **Category**: Type of certificate (Degree, Course, Training, etc.)
- **Issue Date**: Certificate issuance date
- **Expiry Date**: Optional expiration date
- **Additional Details**: Custom fields based on template

**Advanced Options:**
- **Certificate Template**: Use pre-designed templates
- **Custom Fields**: Add organization-specific information
- **Verification Level**: Set verification requirements
- **Privacy Settings**: Control certificate visibility

**Issuance Process:**
1. Complete certificate form with all required details
2. Review certificate preview
3. Confirm issuance
4. **Success**: Certificate issued and confirmation sent
5. **Failure**: Error message with resolution steps

**Bulk Issuance:**
- **CSV Upload**: Issue multiple certificates at once
- **Template Application**: Apply same template to multiple recipients
- **Batch Processing**: Queue large issuance jobs
- **Progress Tracking**: Monitor bulk issuance status

### 3. Read Issued Certificates

![Issued Certificates](https://images.unsplash.com/photo-1586953208448-b95a79798f07?w=1000&h=600&fit=crop&crop=center)

**Certificate Management:**
- **Complete List**: All certificates issued by organization
- **Search & Filter**: Find specific certificates quickly
- **Sort Options**: By date, recipient, status, type
- **Export Options**: Download certificate data

**Certificate Display Information:**
- **Certificate Title**: Name of the issued certificate
- **Recipient**: User who received the certificate
- **Issue Date**: When certificate was created
- **Status**: Active, Expired, Revoked, Pending
- **Verification**: Blockchain verification status
- **Last Activity**: Recent certificate activity

**Certificate Actions:**
- **View Details**: Full certificate information
- **Edit Certificate**: Modify certificate details (if allowed)
- **Revoke Certificate**: Cancel certificate validity
- **Resend**: Send certificate notification again
- **Download**: Get PDF or JSON version
- **Share**: Generate public verification links

**Status Management:**
- **Active Certificates**: Currently valid certificates
- **Pending Certificates**: Awaiting recipient acceptance
- **Expired Certificates**: Past expiration date
- **Revoked Certificates**: Cancelled by issuer
